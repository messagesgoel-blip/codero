# Roadmaps

## Precedence

1. `agent-task-execution-roadmap.md`
   Canonical next-task and execution-sequencing roadmap for contributors and
   shared continuity.
2. `../roadmap.md`
   Roadmap index plus strategic/release context. Useful for phases, proving
   work, and deferred scopes, but not the next-task ledger.
3. `*-backlog.md`
   Candidate backlog only. These files do not override the execution roadmap
   unless a PR explicitly promotes them.
4. `codero-roadmap-v4.md`, `codero-roadmap-v5.md`
   Historical snapshots retained for reference.
